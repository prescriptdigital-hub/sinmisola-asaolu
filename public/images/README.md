# Image guide

Drop your photos into this folder using the exact filenames below and they
will appear automatically. Until a file exists, the site shows an elegant gold
placeholder, so nothing ever looks broken.

Use JPG for photographs (sRGB colour, quality around 80, aim for under 400 KB
each). Use transparent PNG where noted.

| Filename | Page | Size (px) | Notes |
|----------|------|-----------|-------|
| hero-portrait.png | Home hero | 1600 x 2000 | Portrait, transparent background ideal |
| about-portrait.jpg | About | 1200 x 1500 | 4:5 portrait |
| signature.png | (optional) | 700 x 240 | Transparent signature |
| foundation-banner.jpg | Foundation | 1200 x 1500 | 4:5 |
| collection-1.jpg ... collection-6.jpg | Collections | 1000 x 1250 | 4:5 each |
| store.jpg | Collections | 1200 x 900 | The physical shop |
| gallery-1.jpg ... gallery-12.jpg | Gallery | 1200 x 1200 (square), some 1000 x 1400 | Mixed |
| blog-1.jpg ... blog-4.jpg | Blog | 1200 x 800 | 3:2 |
| og-image.jpg | Social sharing | 1200 x 630 | Link preview thumbnail |

To add more gallery or collection photos, edit the arrays in
`app/gallery/page.jsx` and `app/collections/page.jsx`.
